XAUUSD Algorithmic Trading Engine

Built a production-grade high-frequency trading system for XAUUSD (Gold). The core is a custom Native TCP bridge between Python and MetaTrader 5 using persistent read buffers and newline-delimited packet framing — eliminating partial-read errors in a live tick stream processing 1,000–3,000 ticks per minute.

The statistical arbitrage engine triggers trades only when tick density crosses μ + 1.5σ (top ~7% of market events) AND directional order-flow imbalance confirms buyer/seller conviction. No chart patterns, no ML — pure real-time statistical decision-making. Session-level regime constants (μ, σ) are derived offline from 30–90 days of historical tick data per trading session (London, NY, Tokyo).

Supabase PostgreSQL "flight recorder" with automatic fallback to in-memory arrays during network failures. 10-second MT5 position sync protocol eliminates ghost trades. Docker-deployed with custom DNS hardening. Gemini 3 Flash generates AI trade rationale in a Next.js 15 Mission Control dashboard with live price tickers.

Python FastAPI MQL5 / MetaTrader 5 Next.js 15 Gemini API Supabase PostgreSQL Docker TCP Sockets Pandas / NumPy
